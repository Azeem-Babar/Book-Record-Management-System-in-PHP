How to run the Book Record Managment System(BRMS) Project

1. Download the zip file

2. Extract the file and copy brm_db folder

3.Paste inside root directory(for xampp xampp/htdocs, for wamp wamp/www, for lamp var/www/html)

4. Open PHPMyAdmin (http://localhost/phpmyadmin)

5. Create a database with name brm_db

6. Import brm_db.sql file(given inside the zip package in SQL file folder)
